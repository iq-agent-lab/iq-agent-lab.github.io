◈ BUILD METHOD

Study, Constrain, Generate, Validate, Curate

도메인이 텍스트든 학습 자동화든, 같은 운영 사이클 위에서 도구가 자랍니다.

01 · STUDY
도메인 분석
사용 패턴과 실패 지점을 먼저 수집해 도구가 해결할 경계를 정합니다.
02 · CONSTRAIN
규칙을 코드로
출력 형식, 품질 기준, 복구 가능한 에러를 명시적인 제약으로 고정합니다.
03 · GENERATE
AI 양산
번역, 회고, 포스트 변환처럼 반복 가능한 작업을 스트리밍으로 실행합니다.
04 · VALIDATE
검증과 복구
Zod schema, Accepted 확인, 친절한 에러와 자동 복구로 실패를 운영 데이터로 바꿉니다.
05 · CURATE
사람의 최종 판단
AI는 일관성을 만들고, 사람은 방향과 품질을 보증합니다.

iq-leetbuddy는 이 사이클이 데스크톱 환경에서도 작동한다는 첫 사례입니다. LeetCode Accepted 확인, streaming 번역과 회고, draft 저장, atomic commit, root README 인덱스 갱신까지 하나의 로컬 에이전트 루프로 묶었습니다.
